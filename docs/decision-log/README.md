# Decision Log

Use this folder for Architecture Decision Records (ADRs).

## Naming Convention
- `0001-short-title.md`
- `0002-short-title.md`

## Process
1. Copy `/.github/instructions/adr-template.md`.
2. Fill all sections and set status to `Proposed`.
3. Review with Tech Lead.
4. Mark as `Accepted` after approval.
